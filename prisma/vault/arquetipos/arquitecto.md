# Arquitecto

Encuentra propósito en levantar algo que dure: una empresa, una doctrina,
un edificio, una escuela. No le basta con sobrevivir; quiere dejar
huella tangible. Inventores, pioneros, fundadores y empresarios
visionarios encajan en este molde.

**Recuperación de Fuerza de Voluntad.** Cuando crea o establece algo de
valor duradero para quienes vendrán después.
