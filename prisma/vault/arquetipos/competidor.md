# Competidor

Entiende la existencia como una sucesión de pruebas a superar. Cada
situación es una oportunidad para destacar como el mejor, el más
productivo o el imprescindible. Atletas, ejecutivos ambiciosos e
investigadores apasionados encajan aquí.

**Recuperación de Fuerza de Voluntad.** Cuando consigue la victoria
en una prueba relevante. Victorias especialmente difíciles pueden
otorgar puntos adicionales.
