# Arrojado

**Mérito · Físico · 3 puntos**

Encajas bien bajo presión real. En acciones genuinamente arriesgadas
sumas dos dados extra a tu reserva y puedes negar el efecto de un
"dado de fracaso". La dificultad mínima para que aplique debe ser 8 y,
si fallas, puedes sufrir hasta tres niveles de daño según juzgue el
Narrador.
