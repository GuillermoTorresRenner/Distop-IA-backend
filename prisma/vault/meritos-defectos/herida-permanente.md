# Herida permanente

**Defecto · Físico · 3 puntos**

El Abrazo no logró sanar del todo las heridas mortales que sufriste.
Cada noche te despiertas con el nivel de salud Herido, aunque puedes
invertir puntos de sangre para curarte temporalmente.
