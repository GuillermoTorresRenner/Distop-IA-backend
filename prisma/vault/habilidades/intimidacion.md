# Intimidación

Imponerte por miedo, sea con la voz, la postura o un acto. No es ser
amenazante a secas: es lograr que el otro ceda.

**Uso típico:** Fuerza + Intimidación para amenaza física. Manipulación
+ Intimidación para chantaje sutil. Carisma + Intimidación para imponer
respeto sin levantar la voz. Base de varios poderes de **Dominación** y
**Presencia**.
