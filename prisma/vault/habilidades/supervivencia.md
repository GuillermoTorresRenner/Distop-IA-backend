# Supervivencia

Vivir en la intemperie, rastrear, identificar plantas, buscar agua y
refugio. También resistir condiciones extremas con conocimiento.

**Uso típico:** Resistencia + Supervivencia para soportar el ambiente.
Percepción + Supervivencia para rastrear. Base de algunos poderes de
**Protean**.
