# Artesanía

Crear y reparar con las manos: carpintería, herrería, mecánica fina,
electrónica artesanal. Especialízate (carpintería, joyería, etc.) para
obras más exigentes.

**Uso típico:** Destreza + Artesanía para trabajo de precisión.
Inteligencia + Artesanía para diseñar antes de construir.
