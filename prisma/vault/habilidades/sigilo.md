# Sigilo

Moverte sin ser visto u oído. Acechar, esconderte, deslizarte por
pasillos. Junto con **Ofuscación**, base del depredador silencioso.

**Uso típico:** Destreza + Sigilo para colarse. Astucia + Sigilo para
mantener cubierto bajo presión repentina.
