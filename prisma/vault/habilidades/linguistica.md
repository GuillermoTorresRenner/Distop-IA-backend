# Lingüística

Cada personaje arranca con su idioma materno gratis. Para hablar otras
lenguas, modernas o antiguas, se necesita esta habilidad. En niveles
altos también aporta conocimiento general de las estructuras del
lenguaje: reconocer acentos, resolver cifrados, descifrar
inscripciones.

**Uso típico.** Inteligencia + Lingüística para traducir textos
desconocidos. Percepción + Lingüística para captar matices de
acento o detectar un mensaje cifrado.

**Especialidades sugeridas:** lenguas romances, kanji, jeroglíficos,
expresión escrita, cifrados.

**Escala de pericia.**

- 1 — Estudiante: un idioma adicional.
- 2 — Licenciado: dos idiomas adicionales.
- 3 — Posgraduado: cuatro idiomas adicionales.
- 4 — Doctorado: ocho idiomas adicionales.
- 5 — Erudito: dieciséis idiomas adicionales.

**Perfiles típicos:** diplomáticos, embajadores, viajeros, Vástagos
antiguos, criptólogos, eruditos, traductores.
