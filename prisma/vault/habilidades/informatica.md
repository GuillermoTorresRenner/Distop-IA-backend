# Informática

Computadoras, redes, software, hacking básico. A niveles altos: forense
digital, intrusión, escritura de exploits.

**Uso típico:** Inteligencia + Informática para programar, hackear,
recuperar datos. Astucia + Informática para defender un sistema en vivo.
