# Atletismo

Cualquier proeza física que no caiga bajo Pelea o Esquivar: correr,
saltar, escalar, lanzar, nadar. La aptitud bruta del cuerpo entrenado.

**Uso típico:** Destreza + Atletismo para saltos, escalada, esquivar
proyectiles. Fuerza + Atletismo para lanzamientos.
