# Expresión

El arte de comunicar con poder: oratoria, escritura, poesía, canto.
Mover a otros con palabras o gesto artístico.

**Uso típico:** Carisma + Expresión para discursos públicos. Inteligencia
+ Expresión para escribir un texto persuasivo o panfleto.
