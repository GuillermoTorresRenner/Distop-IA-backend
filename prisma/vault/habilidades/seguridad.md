# Seguridad

Familiaridad con las herramientas y técnicas necesarias para forzar
todo lo que pretende mantener algo cerrado o vigilado: cerraduras
mecánicas y electrónicas, alarmas, cámaras, cajas fuertes, puentes
de vehículos. También sirve para el oficio inverso —diseñar e
instalar sistemas— y para reconstruir cómo entró un intruso.

**Uso típico.** Destreza + Seguridad para forzar una cerradura o
hacer un puente. Inteligencia + Seguridad para planear un golpe,
auditar un perímetro o desactivar un sistema complejo.

**Especialidades sugeridas:** cajas fuertes, puentes, alarmas
eléctricas, placas de presión, vehículos.

**Escala de pericia.**

- 1 — Novato: abres una cerradura sencilla.
- 2 — Practicante: sabes hacer un puente.
- 3 — Competente: superas las alarmas domésticas.
- 4 — Experto: puedes abrir una caja fuerte.
- 5 — Maestro: capaz de sacar (o meter) una bomba del Pentágono.

**Perfiles típicos:** ladrones, consultores de seguridad, policías.
