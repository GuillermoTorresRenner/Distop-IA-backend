# Seguridad

Forzar y burlar todo lo que pretende mantener algo cerrado o vigilado:
cerraduras mecánicas y electrónicas, alarmas, cámaras, cajas fuertes,
sistemas de control de acceso. También cubre el oficio inverso:
instalar y diseñar esos mismos sistemas.

**Uso típico:** Destreza + Seguridad para forzar una cerradura.
Inteligencia + Seguridad para planear un golpe o estudiar el perímetro
de un objetivo.
