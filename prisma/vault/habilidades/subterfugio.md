# Subterfugio

Engaño cara a cara: actuar, mentir convincentemente, disimular,
manipular detalles del lenguaje corporal.

**Uso típico:** Manipulación + Subterfugio para mentiras complejas.
Base de varios poderes de **Ofuscación** activa.
