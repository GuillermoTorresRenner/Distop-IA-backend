# Trato con animales

Calmar, dirigir y entender a bestias. Cuanto más alta, mejor "lees" a
los animales y mejor responden a ti.

**Uso típico:** Carisma + Trato con animales para domesticar o convencer.
Manipulación + Trato con animales para imponer una orden contra el
instinto. Base de varios poderes de **Animalismo**.
