# Investigación

Buscar información de forma metódica: archivos, indicios, testigos,
cabos sueltos. La habilidad del detective y del erudito que escarba.

**Uso típico:** Percepción + Investigación para procesar escenas.
Inteligencia + Investigación para reconstruir lo ocurrido a partir de
piezas.
