# Conciencia

Sensibilidad a lo sobrenatural: poderes activos, presencia de otros
seres, residuos psíquicos. Es el "sentido arácnido" de los vástagos.

**Uso típico:** Percepción + Conciencia para detectar uso de Disciplinas,
auras anómalas, lugares cargados.
