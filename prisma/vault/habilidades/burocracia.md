# Burocracia

Navegar instituciones: gobiernos, corporaciones, organismos. Saber a
quién llamar, qué formulario llenar, dónde está el atajo legal.

**Uso típico:** Inteligencia + Burocracia para conseguir documentos
oficiales. Manipulación + Burocracia para sortear trámites con prisa.
