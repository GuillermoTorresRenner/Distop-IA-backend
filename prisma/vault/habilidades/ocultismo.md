# Ocultismo

Conocimiento de lo sobrenatural: tradiciones, rituales, criaturas, mitos.
Esto incluye datos verdaderos **y** las leyendas que los rodean.

**Uso típico:** Inteligencia + Ocultismo para identificar criaturas,
preparar contramedidas, descifrar grimorios. Base de toda
**Taumaturgia**.
