# Empatía

Leer las emociones ajenas: dolor, miedo, deseo, mentira. No el contenido
de los pensamientos, sino el estado emocional debajo.

**Uso típico:** Percepción + Empatía para detectar mentiras, leer ánimo
de una sala. Carisma + Empatía para consolar y conectar.
