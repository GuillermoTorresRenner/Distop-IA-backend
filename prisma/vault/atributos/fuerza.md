# Fuerza

La capacidad bruta para mover, levantar, romper y golpear. La Fuerza mide
el poder muscular del personaje y se aplica a casi todo daño cuerpo a
cuerpo: cuando alzas un peso, derribas una puerta o estrellas un puño
contra la mandíbula de otro vástago.

**Aplicación habitual**

- Levantar, empujar o arrastrar objetos pesados.
- Daño base de armas cuerpo a cuerpo (`damageBase: STRENGTH`).
- Ataques sin armas (Pelea).

**Niveles de referencia (V20)**

| Puntos | Significado                                         |
| ------ | --------------------------------------------------- |
| ●      | Pobre — puedes levantar 18 kg.                      |
| ●●     | Promedio — 45 kg.                                   |
| ●●●    | Buena — 110 kg.                                     |
| ●●●●   | Excepcional — 180 kg.                               |
| ●●●●●  | Sobrehumana para un mortal — 270 kg, parte huesos.  |

> *La sangre puede llevar la Fuerza más allá del límite mortal de forma
> temporal. Combina con Potencia para entrar en territorio sobrenatural.*
